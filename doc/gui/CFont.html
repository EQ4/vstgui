CFont

#include <vstgui.h>

enum CFont;

CFont is an enum with specific values for each font's type :

value size in pixel version
kSystemFont 12 1.0
kNormalFontVeryBig 18 1.0
kNormalFontBig 14 1.0
kNormalFont 12 1.0
kNormalFontSmall
10 MOTIF, MAC
11 WINDOWS
1.0
kNormalFontSmaller
9 MOTIF, MAC
10 WINDOWS
2.0
kNormalFontVerySmall
8 MOTIF, MAC
9 WINDOWS
1.0
kSymbolFont
12 MOTIF, MAC
13 WINDOWS
1.0

Example :

CParamDisplay *myParamDisplay
    = new CParamDisplay (...);
myParamDisplay->setFont (kSystemFont);
