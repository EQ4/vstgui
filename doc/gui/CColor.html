CColor

#include <vstgui.h>

struct CColor;

Structure used for the definition of a color. It is defined by 4 unsigned char values : red, green, blue, unused(alpha).

The predefined variable kTransparentCColor (set to the white color (255, 255, 255)) is used as transparency color for the pixmap.

There are some other predefined colors :

CColor red green blue unused(alpha)
kTransparentCColor 255 255 255 255
kBlackCColor 0 0 0 255
kWhiteCColor 255 255 255 255
kGreyCColor 127 127 127 255
kRedCColor 255 0 0 255
kGreenCColor 0 255 0 255
kBlueCColor 0 0 255 255
kYellowCColor 255 255 0 255
kCyanCColor 255 0 255 255
kMagentaCColor 0 255 255 255

Example :

CColor blackColor (0, 0, 0, 255);
CColor whiteColor (255, 255, 255,255);
CColor greenColor (0, 255, 0, 255);
CColor redColor (255, 0, 0, 255);
