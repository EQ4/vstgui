CButton

#include <vstgui.h>

enum CButton;

CButton is an enum with specific values for each pressed button :

value description
kLButton Mouse button 1 (only)
kMButton Mouse button 2 (only)
kRButton Mouse button 3 (only)
kShift [Shift]-key
kControl
Mac : [Command]-key
PC : [Control]-key
kMenu Menu (Macintosh only)
kAlt [Alt]-key
kApple [Apple]-key (Macintosh only).
kControl|kLButton [Control]-key + mouse button 1,
kLButton|kMButton for button 1+button 2,
kShift|kControl|kLButton [Shift]-key + [Control]-key + mouse button 1

Note :

On pre Mac OS X, the button value is kLButton.
